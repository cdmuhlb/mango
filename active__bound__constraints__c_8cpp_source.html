21 // Minimize f = (x[0] - x[1]) ^ 2 + x[1] ^ 2
22 // subject to
23 // x[0] in [2, 4]
24 // x[1] in [-3, 1]
25 //
26 // The solution is (x[0], x[1]) = (2, 1), and at this point f = 2.
27 //
28 // This script also demonstrates set_relative_bound_constraints().
